R MarkdownでWebサイト生成しよう

lesson: 0 専用のプロジェクトを準備

  1. プロジェクトを新規で準備
  2. 準備したら，Tools - Project Option…を開く
    (もしくは*.RprojファイルをRStudio上で開く)
    1. Build Toolsに進み，Websiteを選択
    2. サイトのルートディレクトリを設定
      (ここではプロジェクトのルートディレクトリにします)
  3. OKをクリック
